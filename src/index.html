An Open Source Minecraft launcher with the ability to manage multiple instances, accounts and mods.

Features

Modpack management
Install and launch modpacks easily, while keeping them clean. No longer do you need to manually install them. OwlixLauncher can do that for you, without installing tons of separate launchers.
Download & Update Mods
Are you sick of finding and adding that one mod you need? OwlixLauncher is able to download & update individual mods from outside modpacks using Modrinth, CurseForge, Technic and more.
Minecraft instance management
Did your settings ever reset after launching an older version? Did you ever accidentally break a world because you opened it in an old version? Are you tired of manually switching mods for different versions, installing modloaders?
OwlixLauncher can help. Each Minecraft instance has its own folder, with separate mods, resourcepacks and other things.
Lightweight & Customizable
You don't need an embedded web browser for a game launcher. OwlixLauncher uses the Qt toolkit, which requires little system resources.
